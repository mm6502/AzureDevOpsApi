# Use-ApiVersion

## SYNOPSIS
Gets the ApiVersion to use for given Azure DevOps collection URI.
If the ApiVersion is not determined, it will default to '5.0'

## SYNTAX

```
Use-ApiVersion [[-Value] <Object>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Gets the ApiVersion to use for given Azure DevOps collection URI.
If the ApiVersion is not determined, it will default to '5.0'
